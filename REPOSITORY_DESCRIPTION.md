# 仓库简介 / Repository Description

股票财报可视化看板：一个面向投研、财务分析和公司研究的 React/Vite 前端模板。它把利润表、现金流、资产负债、关键指标、股东回报和分析观点组织成可阅读的仪表盘页面，适合改造成上市公司研究页、财经内容页、CFO 看板或投研 Memo 工具。

Stock financial dashboard: a React/Vite frontend template for investment research, financial analysis, and company research. It turns income statements, cash flow, balance sheets, key metrics, shareholder returns, and commentary into a readable dashboard interface.

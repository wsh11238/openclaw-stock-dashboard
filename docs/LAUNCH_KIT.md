## V2EX / 分享创造

标题：

```text
开源了一个可直接 Fork 的股票财报可视化看板，真实中文界面 + Demo GIF + 中英文 README
```

正文：

```text
最近把一个真实项目清理成了公开仓库，去掉了密钥、私有域名和部署痕迹，只保留可复用的前端结构、真实中文截图和 Demo GIF。

它主要解决的问题是：把上市公司财报变成可阅读、可复用的 React/Vite 投研仪表盘。

适合这些场景：
- 上市公司研究页
- 财经内容页
- CFO 看板
- 投研 Memo
- 财报可视化组件学习

Fork 后主要替换 财务指标常量和图表数据，就可以改成自己的项目。

GitHub: https://github.com/wsh11238/openclaw-stock-dashboard
```
